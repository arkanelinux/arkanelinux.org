Embrace the Arkane

Arkane Linux is an opinionated, immutable multi-root Arch-based distibution which aims to provide a GNOME-centered experienced with minimal yet full featured and sensible non-intrusive defaults.

Sensible defaults

Arkane Linux avoids unnecessary and intrusive functionality and configuration, it provides a stock gnome-shell bundled exclusively with core apps required to offer a minimal yet full-featured GNOME desktop experience. Every effort is made to debloat and streamline there where deemed required without affecting functionality.

Multi-root architecture

Did an upgrade introduce a bug or some other issue? Just roll back to the previous version, it is probably still installed, and if not you can freely pull it again from the repos.

Seamless and consistent

From the installation to the initial setup and beyond, everything is done to ensure uniformity, applications or configuration which break said uniformity are actively barred on a new Arkane Linux installation.

Tailing the bleeding edge

By default Arkane Linux is configured with and to make use of currently relevant and/or upcoming technologies and software.

Install once, update forever

At its core Arkane Linux is still Arch Linux allowing for roling update. When a new major release of Arkane is made available you don't have to upgrade to said new release, you can just stick to your current familiar configuration.

Embrace the Arcane

Arkane Linux is an opinionated soft-fork of Arch Linux, it aims to be an accessible GNOME-centered experience and with sensible non-intrusive defaults. The system is optimized to maximize performance on modern hardware by targeting the x86-64-v3 CPU microarchitecture level.

Sensible defaults

Arkane Linux avoids unnecessary and intrusive functionality and configuration, it provides a stock gnome-shell bundled exclusively with the core apps required to offer a minimal yet fully-featured GNOME desktop experience.

No potential untapped

Arkane Linux is a full recompilation of Arch Linux for modern CPU microarchitecture levels, squeeze every bit of performance out of your modern x86-64-v3 capable hardware.

Seamless and consistent

From the installation to the initial setup and beyond, everything is done to ensure uniformity, applications or configuration which break said uniformity are actively barred on a new Arkane Linux installation.

Eyes towards the future

By default Arkane Linux is configured with and to make use of currently relevant and/or upcoming technologies and software.

Install once, update forever

At its core Arkane Linux is still Arch Linux. When a new major release of Arkane is made available you don't have to upgrade to said new release, you can just stick to your current familiar configuration. Also if CPU microarchitecture levels are ever dropped you could always switch the system over to a normal Arch Linux one.
